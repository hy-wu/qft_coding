\section{引言}

量子場論在高溫下的行為是夸克-膠子等離子體（QGP）唯象學的理論基礎。
有限溫場論的核心工具之一是松原（Matsubara）形式——
將溫度引入場論的途徑是將時間方向緊緻化為半徑 $\beta = 1/T$ 的圓，
粒子的統計性質則體現在松原頻率的離散求和當中\footnote{詳細介紹見文獻~[1]~第~3~章、[2]~第~2~章、[3]~第~1~章。}。

然而，絕大多數教科書的推導止於解析結果的封閉形式。
對於實際的 QGP 研究——例如計算重夸克擴散係數、夸克偶素的譜函數——
我們最終需要\textbf{數值地}做松原和、動量積分和解析延拓。
本文的目的是填補「課本公式」與「可執行代碼」之間的 gap。

我們選擇最簡單的 $\phi^4$ 理論作為 playground，因為：
\begin{itemize}[nosep]
\item 單圈計算可解析做，方便與數值結果交叉驗證；
\item 譜函數的溫度展寬類比於 QGP 中的夸克偶素溶解；
\item 計算代價低，普通筆記型電腦即可在數秒內完成。
\end{itemize}

\section{有限溫標量場論基礎}

\subsection{松原形式}

在有限溫度 $T$ 下，場論的配分函數可寫為虛時路徑積分：
\begin{equation}
Z = \int \mathcal{D}\phi \; \exp\left[-\int_0^\beta d\tau \int d^3\mathbf{x}\; \mathcal{L}_E\right],
\label{eq:path-integral}
\end{equation}
其中 $\beta = 1/T$，歐幾里得拉氏量為
\begin{equation}
\mathcal{L}_E = \frac{1}{2}(\partial_\tau\phi)^2 + \frac{1}{2}(\nabla\phi)^2 + \frac{1}{2}m_0^2\phi^2 + \frac{\lambda}{4!}\phi^4.
\label{eq:lagrangian}
\end{equation}
場 $\phi(\tau, \mathbf{x})$ 在虛時方向上滿足（玻色子的）週期邊界條件：
$\phi(0, \mathbf{x}) = \phi(\beta, \mathbf{x})$\footnote{費米子對應反週期邊界條件。}。

對 $(\ref{eq:path-integral})$ 做微擾展開，自由傳播子在松原頻率空間中為
\begin{equation}
\Delta(i\omega_n, \mathbf{k}) = \frac{1}{\omega_n^2 + \mathbf{k}^2 + m_0^2},
\label{eq:propagator}
\end{equation}
其中 $\omega_n = 2\pi n T$（$n \in \mathbb{Z}$）是玻色子的松原頻率。

\subsection{Tadpole 自能}

在 $\phi^4$ 理論中，二點函數的單圈修正由 tadpole 圖給出：
\begin{equation}
\Pi = \frac{\lambda}{2}\; T\sum_{n=-\infty}^{\infty} \int \frac{d^3\mathbf{k}}{(2\pi)^3}\; \Delta(i\omega_n, \mathbf{k}).
\label{eq:tadpole}
\end{equation}
這裡的因子 $1/2$ 來自對稱因子，$T\sum_n$ 是松原和。

松原和是解析可做的。利用標準技巧\footnote{見~[1]~附錄~A 或~[3]~第~2.2~節。}：
\begin{equation}
T\sum_{n=-\infty}^{\infty} \frac{1}{\omega_n^2 + E_{\mathbf{k}}^2}
= \frac{1}{2E_{\mathbf{k}}}\bigl[1 + 2n_B(E_{\mathbf{k}})\bigr],
\label{eq:matsubara-sum}
\end{equation}
其中 $E_{\mathbf{k}} = \sqrt{\mathbf{k}^2 + m_0^2}$，$n_B(E) = 1/(e^{\beta E} - 1)$
為玻色-愛因斯坦分佈。

將 (\ref{eq:matsubara-sum}) 代入 (\ref{eq:tadpole})，自能分解為零溫真空部分和溫度部分：
\begin{equation}
\Pi = \Pi_{T=0} + \Pi_{\text{th}}(T).
\label{eq:pi-split}
\end{equation}
真空部分是發散的，需要在重整化過程中減除。
溫度部分是有限的，其表達式為
\begin{equation}
\Pi_{\text{th}}(T) = \frac{\lambda}{2} \int \frac{d^3\mathbf{k}}{(2\pi)^3}\; \frac{n_B(E_{\mathbf{k}})}{E_{\mathbf{k}}}.
\label{eq:pi-th}
\end{equation}
這就是我們將要數值計算的目標——它正是熱質量 $m_{\text{th}}^2(T)$ 的來源。

\section{練習一：熱質量的數值計算}

\subsection{從解析推導到數值積分}

將 (\ref{eq:pi-th}) 寫為球坐標形式：
\begin{equation}
\Pi_{\text{th}}(T) = \frac{\lambda}{4\pi^2} \int_0^{\infty} dk\; k^2\; \frac{n_B(E_k)}{E_k},
\label{eq:pi-th-spherical}
\end{equation}
其中 $E_k = \sqrt{k^2 + m_0^2}$。

這是個一維積分，適用高斯求積。積分核隨 $k$ 的衰減由 $n_B(E_k)$ 控制：
在 $k \gg T$ 時，$n_B \sim e^{-k/T}$，故截斷到 $k_{\max} \sim 20T$ 已足夠精確。

\subsection{Python 實現}

對應的程式碼在 \texttt{01\_tadpole\_thermal\_mass.py} 中\footnote{全文見 GitHub: \url{https://github.com/hy-wu/qft_coding}。}。

核心是玻色-愛因斯坦分佈函數：
\begin{verbatim}
def bose_einstein(E, T):
    if T <= 0:  return 0.0
    if E / T > 100:  return 0.0  # avoid overflow
    return 1.0 / (np.exp(E / T) - 1.0)
\end{verbatim}
注意 \texttt{exp(E/T)} 在 $E/T > 100$ 時超出雙精度浮點範圍，需提前截斷。

被積函數與數值積分：
\begin{verbatim}
def tadpole_integrand(k, T):
    Ek = np.sqrt(k**2 + M0**2)
    return 4.0 * np.pi * k**2 * bose_einstein(Ek, T) / Ek

def thermal_self_energy(T):
    cutoff = max(20.0 * T, 10.0)
    result, _ = integrate.quad(tadpole_integrand, 0, cutoff,
                                args=(T,))
    return LAMBDA / 2.0 * result / (2 * np.pi)**3
\end{verbatim}
\texttt{integrate.quad} 是 SciPy 的自適應高斯求積，它負責處理積分核在 $k \to 0$ 時的潛在奇異性。

\subsection{高溫極限}

當 $T \gg m_0$ 時，可以忽略質量並解析算出 (\ref{eq:pi-th})：
\begin{equation}
m_{\text{th}}^2(T) \equiv \Pi_{\text{th}}(T) \;\xrightarrow{T \gg m_0}\; \frac{\lambda T^2}{24}.
\label{eq:ht-limit}
\end{equation}
這是有限溫場論中最經典的結果之一——標量場在高溫下獲得與 $T^2$ 成正比的熱質量。
在 QCD 中，對應的是德拜質量 $m_D \sim gT$\footnote{見~[1]~第~5.5~節或~[3]~第~4.3~節。}。

\section{練習二：譜函數與溫度展寬}

熱質量只是故事的一半。在 QGP 中，粒子不僅獲得有效質量，還因與熱浴的碰撞而具有有限壽命，表現為譜函數的展寬。
這正是夸克偶素在 QGP 中「溶解」的微觀機制\footnote{詳見~[4]~以及~[5]。}。

\subsection{推遲傳播子與譜函數}

推遲傳播子的定義為\footnote{見~[1]~第~3.4~節或~[3]~第~2.4~節。}
\begin{equation}
D_R(\omega, \mathbf{p}) = \frac{1}{-\omega^2 + \mathbf{p}^2 + m_0^2 + \Pi_R(\omega, \mathbf{p})},
\label{eq:retarded}
\end{equation}
其中 $\Pi_R$ 是推遲自能。譜函數則為
\begin{equation}
\rho(\omega, \mathbf{p}) = -\frac{1}{\pi}\, \operatorname{Im} D_R(\omega, \mathbf{p}).
\label{eq:spectral}
\end{equation}
在自由粒子極限下，$\rho(\omega, \mathbf{p}) = \operatorname{sgn}(\omega)\,\delta(-\omega^2 + \mathbf{p}^2 + m_0^2)$。

\subsection{熱寬度的唯象引入}

在 $\phi^4$ 理論的高溫極限，$2 \to 2$ 散射過程賦予準粒子一個衰變寬度\footnote{見~[6]~以及~[7]~中有關標量場阻尼率的討論。}：
\begin{equation}
\Gamma(T) \sim \frac{\lambda^2 T}{4\pi}.
\label{eq:width}
\end{equation}
在準粒子極限附近，自能可近似為
\begin{equation}
\Pi_R(\omega, \mathbf{p}) \approx m_{\text{th}}^2(T) - i\omega\Gamma(T).
\label{eq:self-energy-approx}
\end{equation}
代入 (\ref{eq:retarded})-(\ref{eq:spectral}) 即得 Breit-Wigner 形式的譜函數：
\begin{equation}
\rho(\omega, \mathbf{p}) = \frac{1}{\pi}\,
\frac{\omega\Gamma}{(\omega^2 - \omega_{\mathbf{p}}^2)^2 + (\omega\Gamma)^2},
\label{eq:breit-wigner}
\end{equation}
其中 $\omega_{\mathbf{p}}^2 = \mathbf{p}^2 + m_0^2 + m_{\text{th}}^2(T)$。

\section{與 QGP 唯象的聯繫}

以上練習從標量場論出發，逐步過渡到 QCD 的 HTL 理論，其物理圖像直接與 QGP 唯象相關：

\begin{itemize}[nosep]
\item \textbf{德拜屏蔽}：QCD 中膠子的熱質量（德拜質量）$m_D^2 = g^2T^2(N_c/3 + N_f/6)$\footnote{見~[1]~第~5.5~節、[3]~第~4.3~節。}。
這控制了誇克-膠子等離子體中色電荷的屏蔽長度 $\lambda_D = 1/m_D$。

\item \textbf{夸克偶素溶解}：$J/\psi$ 和 $\Upsilon$ 在 QGP 中的譜函數與我們計算的標量譜函數定性類似。
當溫度超過某個閾值，束縛態譜峰的展寬超過束縛能，粒子「溶解」\footnote{見~[4]~以及 LHC 上 ALICE、CMS 合作的相關實驗報告。}。
這是 QGP 存在的關鍵實驗信號之一。

\item \textbf{重夸克擴散}：重夸克（粲、底）在 QGP 中的動量擴散係數 $\kappa$，
可透過重夸克譜函數的零能極限提取：$\kappa = \lim_{\omega\to0} 2T\rho(\omega, \mathbf{0})/\omega$\footnote{見~[8]~中有關 $\hat q$ 與 $\kappa$ 的詳細討論。}。
這正是本文可以自然擴展的方向。
\end{itemize}

\section{練習三：HTL 自能與等離激元色散}

前面兩個練習使用的都是 $\phi^4$ 標量理論。但要聯繫實際的 QGP，需要轉向規範理論。
在高溫下，膠子（或光子）的自能有一個重要的非微擾貢獻——Hard Thermal Loop (HTL)\footnote{見~[9]~原始文獻，或~[1]~第~5.5~節的綜述。}。
HTL 自能描述了等離子體中集體激發的傳播，並在 QGP 的唯象學中扮演核心角色。

\subsection{HTL 自能的解析形式}

在 QCD（或 QED）的高溫極限下，膠子的自能可以分解為縱向（電）和橫向（磁）兩個通道。
對於 $N_c$ 種顏色、$N_f$ 種味的 QGP，德拜質量為
\begin{equation}
m_D^2 = g^2 T^2 \left(\frac{N_c}{3} + \frac{N_f}{6}\right).
\label{eq:debye-mass}
\end{equation}

縱向和橫向 HTL 自能的解析表達式分別為\footnote{推導見~[3]~第~4.3~節。}：
\begin{align}
\Pi_L(\omega, p) &= m_D^2 \left[1 - \frac{\omega}{2p} \ln\frac{\omega + p}{\omega - p}\right],
\label{eq:htl-L}\\
\Pi_T(\omega, p) &= \frac{m_D^2}{2} \left[\frac{\omega^2}{p^2} - \frac{\omega(\omega^2-p^2)}{2p^3} \ln\frac{\omega + p}{\omega - p}\right].
\label{eq:htl-T}
\end{align}
這兩個表達式對 $|\omega| > p$ 的傳播區是實函數；在 $|\omega| < p$ 的類空區，
對數函數獲得虛部（$ \ln\frac{\omega+p}{\omega-p} \to \ln\frac{p+\omega}{p-\omega} - i\pi $），
對應著朗道阻尼。

\subsection{色散關係的數值求解}

縱向和橫向傳播子的極點由以下方程給出：
\begin{align}
\text{縱向（等離激元）：}&\quad p^2 + \Pi_L(\omega, p) = 0,
\label{eq:disp-L}\\
\text{橫向：}&\quad -\omega^2 + p^2 + \Pi_T(\omega, p) = 0.
\label{eq:disp-T}
\end{align}

數值求解的關鍵是對每個 $p$ 找到滿足方程的 $\omega > p$。
兩個方程均爲單調函數且在 $p$ 和 $\infty$ 處符號相反，適合用二分法（bisection）求解\footnote{實現見 \texttt{03\_HTL\_plasmon.py} 中的 \texttt{solve\_dispersion} 函數。}。
Python 代碼的核心是對每個 $p$ 調用 \texttt{scipy.optimize.root\_scalar} 進行 bracketed root-finding。

\section{總結與展望}

本文展示了從有限溫場論的基本公式出發，透過約 300 行 Python 程式碼，
完成從松原和到等離激元色散的完整計算。主要收穫：

\begin{enumerate}[nosep]
\item 有限溫度的微擾計算可自然地拆解為「解析推導」和「數值積分/求根」兩步；
\item 熱質量 $m_{\text{th}}^2$ 的數值結果在高溫下精確趨近 $\lambda T^2/24$；
\item 譜函數從自由 $\delta$ 函數演化為高溫下的 Breit-Wigner 形式，
反映了準粒子在熱浴中獲得有效質量與有限壽命的物理圖像；
\item HTL 自能的數值求解揭示了 QGP 中等離激元的色散關係——在 $p=0$ 處簡併為 $\omega_{\text{pl}} = m_D/\sqrt{3}$，
在 $p \to \infty$ 處趨近於光錐；
\item 以上所有結果均直接類比於 QGP 的德拜屏蔽、等離激元激發與夸克偶素溶解。
\end{enumerate}
